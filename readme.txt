== Description ==

A plugin which Create a slider, with fade animation and show title and description with animation. 
<a href="http://dewa.org.pk/expo_Comparison/spyrowebz-slider/" target="_blank">See the demo </a>
You can set the image height and width.
You can hide title, description, arrows and thumbnail.
You can Auto play and stop the slider.
You can stop slide on mouse over.

How To use:-

[youtube https://www.youtube.com/watch?v=iEHq6TsJwvo]


Once you active the plugin you will see a spyrowebz slide in your admin menu on your dashboard normally below the post menu.
click on spyrowebz slider and add slide as much as you want. but keep in mind that your image height and width should be same for each image. for example if your width=1000px and height=400px the it should be for all.
Add title to your slider, add description to your slide and set a feature image to your slide, and if you want to link the slide see on your right side under the publish and put your link there, publish your slide.
Once you add the slides then see the setting below the setting and click it and set the setting. put the orginal image height and width.

Into your page or post put the shortcode [spyrowebz_slider] where you want to show it.

== Installation ==

This section describes how to install the plugin and get it working.


  
   From your WordPress dashboard

    1. Visit 'Plugins > Add New'
    2. Search for 'Spyrowebz Gallery'
    4. Install Spyrowebz Gallery
    3. Activate Spyrowebz Gallery from your Plugins page.
